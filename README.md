## Credits

- Special thanks to [Nuxt](https://nuxt.com/), the [Vue](https://vuejs.org/) framework for bringing the best development experience.
- Serverless hosted on [Cloudflare Workers](https://workers.cloudflare.com/) with [NuxtHub](https://hub.nuxt.com/).
- [Cloudflare D1](https://developers.cloudflare.com/d1/) SQL database, and [Cloudflare Cron Triggers](https://developers.cloudflare.com/workers/configuration/cron-triggers/) for keeping data updated.
- UI powered by [Nuxt UI](https://ui.nuxt.com/).
- [twisted](https://github.com/Sansossio/twisted) package for fetching Riot Games API data.
- [twurple](https://github.com/twurple/twurple) for interacting with Twitch API.
